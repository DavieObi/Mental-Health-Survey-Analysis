# Contributing to Mental Health Survey Analysis

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository.
2. Create a new branch (`git checkout -b feature-name`).
3. Make your changes.
4. Test your updates locally.
5. Commit your changes (`git commit -am 'Add feature'`).
6. Push to your branch (`git push origin feature-name`).
7. Open a Pull Request with a clear description.

## Code of Conduct

Please be respectful and professional when interacting with the community.

## Issues

If you encounter bugs or have feature requests, open an issue with detailed information.
